# Riffs

## Description

Riffs is a robust and free-to-use music-sharing service. Users can upload, comment, and play music.

To learn Vue and Firebase, I took a course on Udemy. I then used the capstone project I built in that course as the blueprint for this app. I plan to continue working on it, adding new features.

I hope you enjoy this project and if you have any questions or comments, feel free to contact me.
